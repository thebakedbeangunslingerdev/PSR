# Python Shells Resurrected (PSR) 
is a shell written in Python (as the name suggests). I created it because I wanted to explore how shells work internally while building something lightweight, customizable, and easy to extend in Python.
## How to use the shell
Once you start using PSR, you might be _clueless_ on how to use it. It **Is not a copy of Bash. It has a whole new set of commands**. **Commands such as ls, cd, cat will not work!**. So here is a small cheat sheet:



                                                    _PSR COMMAND MENU_
`locate`   -> View your current folder path




`peek`     -> See what files are inside this folder



`view`     -> Read and view a file's contents directly



`shift`    -> Move to a different directory




`cleanse`  -> Strip target characters out of a file



`scribe`   -> Create and edit text files




`scribel`  -> Launch a minimal GUI canvas drawing tool



`help`     -> Show this menu




`exit`     -> Close PSR







PSR is still a work in progress.
More Utils and programs will come in the `future`.
